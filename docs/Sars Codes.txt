PENSION FUND CODES
==================

4472 - Employer's Pension Fund Contributions

    - This is the amount paid by the employee towards his/her pension fund.
    - Deduction
    - Is NOT part of taxable income.
    - This code's value must exclude the value from code 4585
    - This code is only applicable from 2017 onwards
    - If this code is completed code 3817 and 4001 must also be completed.
    - Payslip Item Type Code = 2005

3817 - Employer Pension Fund Contributions

    - This is the amount paid by the employer towards the pension fund on the employee's behalf.
    - Fringe Benefit
    - Is part of taxable income
    - Payslip Item Type Code = 4001
    
4001 - Provident Fund Contributions Paid and 'Deemed Paid' by Employee

    - The amounts of code 4472 and 3817 combined.
