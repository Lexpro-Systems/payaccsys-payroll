Added UIF Declaration

Functionality has been implemented to export a UIF declaration which can be electronically submitted as an attachment to the Department of Labour via e-mail to: declarations@labour.gov.za.

Export UIF declaration.

The UIF declartion function can be executed by clicking the 'UIF Declaration' button at the top of the UIF Report panel:

UIF declaration button.

Employee Tags

Employee tags can be accessed via the 'Tags' menu item in the main menu:

Employee Tags.

In the employee tags section, you can add a tag name and link multiple employees to the tag name, you can also edit and remove a tag name.

Add Employee Tags.

Attendance Tags

Once you have added the employee tag names, you can select the tags in the attendance register.

Attendance Tags.

And when you have selected a tag name, you will only see the selected employees on the attendance register screen.

Attendance Employees.
