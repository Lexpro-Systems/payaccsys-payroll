Payment Period End Day

A payment period end day setting has been added to the employee's employment details section. This setting will now be used to calculate the end date of the employee's payslips, rather than payment day, which was previously used.

The payment day setting now only indicate the day on which the employee receives payment.

Note: Existing employees will have their payment period end day set to the same value as payment day. Please update your employee details if this is incorrect.

EMP501 Report

An 'EMP501 Report' has been added to help fascilitate the Employer Reconciliation process. The report shows PAYE, UIF and SDL totals for the relevant months of a given tax year.

EMP501 report.

EMP201 Report Update

The 'EMP201 Report' has been updated to display results for any given month of a tax period to bring it into line with the 'EMP501 Report'.
