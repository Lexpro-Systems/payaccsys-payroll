Overlapping Payrun Periods

Payrun periods will now be allowed to overlap, allowing users to create multiple payruns for the same period.

Birthday Notifications

An option for sending birthday notifications has been added to the 'Miscellaneous' settings of the 'Setup' section. If this option is checked, all employees with valid email addresses will be notified via email of impending birthdays of co-workers. A birthday report has also been added.

Work Schedule
