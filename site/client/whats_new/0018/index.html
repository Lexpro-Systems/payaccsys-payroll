Updated report layout.

Overtime Payslip Item

An overtime payslip item has been added. Overtime will be reflected under code 3607 in tax certificates.

Overtime payslip item.

Overtime Report

Added an overtime report in the "Payrun Reports" section to display the overtime details for each employee in a specified payrun or over a period.

Overtime report.

Bulk Allocate Leave

Functionality has been added to allocate leave to multiple employees at once. Please note that leave is calculated automatically for each employee based on the leave type(s) to which the employee is subscribed. The use of this function to allocate leave manually should be reserved primarily to setup leave for the first time or to make corrections, if required.

Bulk allocate leave.
