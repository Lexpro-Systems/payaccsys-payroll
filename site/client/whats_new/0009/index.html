Self-Service Portal

Employees can now have access to their payslips, view their available leave, and even make leave requests via Lexpro Payroll's self-service portal!

To access the self-service portal, simply point your employees to the web address below, where they will be able to sign-up for an account:

https://www.lexproonline.co.za/payroll/employee/

Leave Requests

We've added a 'Leave Request' section that can be accessed via the main menu. From here you will be able to view and manage all the leave requests made by employees via the self-service portal.

Leave Request Administrator

A setting has been added under the 'Miscellaneous' tab of the 'Setup' section (accessed via the main menu) where you may enter the email address of the person you wish to approve or decline leave requests (this may be useful if, for example, the person you wish to administrate leave requests does not have access to the Lexpro Payroll system).

If an email address is specified, an email will be sent to the aforementioned address the moment leave request is made via the Self-Service Portal, allowing the recipient to approve or decline the request.

'Leave Request' settings.

Employee Leave Report

An 'Employee Leave' report has been added to the 'Reports' section (accessed via the main menu). The report compliments the 'Leave Summary' report by displaying full leave details for a specified employee.

Undo Payrun Processing

You can now undo the processing of a payrun by clicking on the 'Undo Processing' option in the menu () on the right side of the listed payrun.

When the processing of a payrun is undone, all leave earned as well as loan transactions that may have occurred when the payrun was processed will be removed.

'Undo Payrun Processing' menu item.

Set Lexpro Export Posting Date

An option has been implemented to quickly set the posting date for all the listed transactions to be exported to Lexpro Accounting. Click on the menu icon () to the right of the 'Transactions to Post' heading and select the 'Set Posting Date' item to do so.

'Set Posting Date' menu item.

Export Attendance History

Export the attendance history in 'Excel' or 'CSV' format by clicking on the 'Export' button at the top right of the 'Attendance History' page.
