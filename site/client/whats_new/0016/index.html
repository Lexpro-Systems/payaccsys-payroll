Post Salaries to Employee Specific Lexpro Accounts

When posting payrun transactions to Lexpro Accounting users now have the option to post employee salaries to a single account or to specify a dedicated account for each employee.

To use this feature please update your Lexpro Accounting configuration as follows:

  • Select the "Setup" option from the main menu and go to the "Accounting Setup" section.
  • Edit the configuration by clicking on the "Configuration" button or selecting the "Configuration" option from the drop-down menu.
  • Select "No" for the "Use single account for salaries?" option.
  • Select the appropriate Lexpro account for each listed employee.

Return of Earnings (Workmen's Compensation) Report

Refresh Payrun

A button has been added to allow users to refresh payrun calculations. Refreshing payrun calculations will re-calculate all auto-calculated values based on the latest available tax rates and tax calculation method.
