Payslip Setup

A payslip setup section has been added to the setup screen which will allow you to add a logo of your choice to employee payslips. You can resize and reposition the image and display a payslip preview to ensure the logo looks exactly the way you like it.

New payslip setup section.

Add Payslips to Payruns

A menu () has been added to the top right corner of the edit payrun screen with an option to add payslips to the payrun. Payslips that have been deleted can be restored or payslips can be added for new employees or employees whose payment details have changed.

Add payslips screen.

Recreate Payslip Items

A menu option has been has been added to the payslip menu () of every payslip displayed in the edit payrun screen. This option will enable you to create the payslip items from scratch, making certain that the payslip is up-to-date with any changes that might have been made to the employee's earnings / deductions.
